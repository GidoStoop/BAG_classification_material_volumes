## 🌇 Classifying the Dutch building stock and estimate material volumes

This repository contains a Jupiter Notebook file that uses data from Sprecher et al., (2022), BAG (Kadaster, 2025a),Kadaster (2025b) and 3D-BAG (TU Delft, 2025) to classify buildings in the Dutch building stocks and then uses this classification to estimatie the material volumes in the buildings. </br>

- Next steps: Map Future build- and demolition projects to predict material flow
- Use prediction to improve logistics and enable circularity
- Further classify buildings in 'other' category
